Работу документировать в удобном формате
README.md || readme.docx

Задание 1.1
 Открыть терминал
 Выполнить команду git --version
 Убедиться что git установлен. (Сделать скрин)
 Выполнить git config --global user.name "username"
 Выполнить git config --global user.email "your@mail.com"
 Выполнить git config --list. Увидеть свои данные (Сделать скрин)
 
Задание 2.1
 Создать новую папку "about-me" и перейти в нее
 Инициализировать репозиторий командой git init
 Проверить что появилось в папке
 Выполнить команду git status - в ответ должны получить "No commits yet"
 
Задание 2.2
 Сделать файл about_me.txt и написать 3 факта о себе
 Проверить статус через git status
 Добавить файл в "конверт": git add about_me.txt
 Добавление в конверт позволяет отслеживать историю изменений файла
 Снова проверить статус
 Запечатать конверт и подписать: git commit -m "Добави(л/ла) файл о себе"
 Флаг -m это подпись конверта
 Запечатывание конверта позволяет сохранить состояния файла.

Задание 2.3
 Удалить одну строчку из about_me.txt
 git add about_me.txt
 git commit -m "Удалили один факт о себе"
 Проверить историю изменений с помощью: git log
 Можно использовать компактный вид: git log --oneline
 
Задание 3.1
 Войти на github.com
 Нажать на кнопку "New repository" (Зеленая в правом верхнем углу)
 Назвать репозиторий about-me
 Оставить его публичным
 НЕ ставить галочки
 Нажать на "create repository"
 
Задание 3.2 Связываем локальный репозиторий с github
 В терминале убедиться что находимся в папке about-me
 Выполнить команду из инструкции Github: git remote add origin https://github.com/ВАШ-НИК/about-me.git
 git remote -v должен отобразить адрес вашего репозитория
 
 origin - это имя псевдоним для адреса вашего удаленного репозитория.
 
Задание 3.3 Отпраляем историю изменений на github
 Выполнить git push -u origin main
 При запросе авторизации: если не настроен SSH, войти через браузер или ввести токен
 Открыть страничку репозитория на github - Мы должны увидеть свой файл и историю коммитов
 Нажать на "3 commits" на github - посмотреть историю в веб интерфейсе
